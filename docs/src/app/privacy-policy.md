# Privacy Policy

_Last updated: April 2026_

Arc Port is a Chrome extension that ports select features from the Arc browser into Google Chrome. This Privacy Policy explains how the extension handles information in accordance with the [Chrome Web Store User Data Policy](https://developer.chrome.com/docs/webstore/program-policies/user-data-faq).

---

## 1. Data Collection

**Arc Port does not collect, store, or transmit any personal or sensitive user data.**

The extension operates entirely within your local browser environment. No information about you, your browsing activity, or your device is ever sent to any external server, third party, or developer-controlled system.

---

## 2. Permissions Used

Arc Port requests the following Chrome permissions, each used solely to deliver its core functionality:

| Permission | Purpose |
| :--- | :--- |
| `activeTab` | Read the URL of the currently active tab to set or reset a checkpoint |
| `tabs` | Manage tab navigation, detect tab state changes, and restore tabs to their checkpoint |
| `storage` | Persist checkpoint data locally in the browser (Chrome Storage API) |
| `scripting` | Execute in-page scripts required for checkpoint and tab-reset operations |
| `contextMenus` | Add right-click context menu entries for checkpoint operations |
| `tabGroups` | Detect when a tab is added to a group to automatically set a checkpoint |
| `<all_urls>` (host permission) | Allow the extension to operate across all websites, required for tab checkpoint functionality regardless of which page the user has open |

No permission is used to collect, aggregate, profile, or transmit user data. All data generated by these permissions (e.g., saved checkpoint URLs) is stored exclusively in your local browser via `chrome.storage` and never leaves your device.

---

## 3. Local Storage

Checkpoint URLs are stored using the Chrome Storage API (`chrome.storage.local` or `chrome.storage.sync`). This data:

- Remains on your device (or syncs across your own Chrome profile if Sync is enabled by you)
- Is never accessed by the developer
- Can be cleared at any time by removing or disabling the extension

---

## 4. No Data Sharing

Arc Port does **not**:

- Share any data with third parties
- Use or transfer data for advertising, profiling, or analytics
- Allow any human (including the developer) to access your browsing data or stored checkpoints

---

## 5. No Tracking or Analytics

Arc Port contains **no** analytics libraries, tracking pixels, telemetry systems, or remote logging of any kind.

---

## 6. Children's Privacy

Arc Port does not knowingly collect data from users of any age. Since no data is collected at all, the extension does not present privacy risks specific to children.

---

## 7. Changes to This Policy

If this extension is ever updated to handle personal or sensitive user data in the future, this Privacy Policy will be updated accordingly and users will be notified through the extension's release notes.

---

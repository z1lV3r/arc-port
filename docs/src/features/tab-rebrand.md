## Operations

| Operation | Keyboard shortcut | Context menu | UI |
| :---: | :---: | :---: | :---: |
| Show custom tab name editor | `Alt+Shift+E` / `Option+Shift+E` | ✔ | ✔ |
| Set custom tab name | ❌ | ❌ | ✔ |
| Clear custom tab name | ❌ | ❌ | ✔ |
| Show custom tab icon editor | `Alt+Shift+I` / `Option+Shift+I` | ✔ | ✔ |
| Set custom tab icon | ❌ | ❌ | ✔ |
| Clear custom tab icon | ❌ | ❌ | ✔ |
